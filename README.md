# Instagram Clone

Firebase 학습용 인스타그램 클론 앱

- Authorization 구현
- Firebase Store(Realtime DB) 적용

## Installation

npm 으로 Google firebase Cli 설치 및 dependency 설치

```bash
npm install -g firebase-tools

npm install
```

## Usage

npm run build 하여 빌드된 결과물 생성 후, firebase deploy 로 배포

```bash
npm run build

firebase deploy
```
